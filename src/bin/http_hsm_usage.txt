A software not-HSM accessible via HTTP

Usage: http_hsm [OPTIONS] --key <KEY>

Options:
  -l, --listen <LISTEN>        The IP/port to listen on [default: 127.0.0.1:8080]
  -n, --name <NAME>            Name of the hsm in logging [default: hsm{listen}]
  -k, --key <KEY>              Derive realm key from this input
  -s, --state-dir <STATE_DIR>  Directory to store the persistent state file in [default: a random temp dir]
  -h, --help                   Print help
